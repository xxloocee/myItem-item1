# 收录管理

Project address: https://github.com/xxloocee/myItem.git

```bash
# clone the project
git clone 

# enter the project directory
cd myItem/item1

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:8080

## Build

```bash
# build for production environment
npm run build
```
